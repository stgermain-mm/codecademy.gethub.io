HTML Reference

DOCUMENT STRUCTURE:

TAG DESCRIPTION
<html> Defines the root of an HTML document.
<head> Contains meta data about the HTML document.
<title> Sets the title of the HTML document (shown in the browser's title bar or tab)
<body> Contains the content of the HTML doc / info.

TEXT MARKUP:

TAG DESCRIPTION
<h1>to<h6> Headings (from largest to smallest)
<p> contains a paragraph worth of info
<br> Causes the line to 'break' and start a new one.
<strong> or <b> causes the text to be 'bold'
<em>or <i> Italic's the text

Table Tags

TAG NAME DESCRIPTION
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers.
<tbody> Table Head The set of rows containing actual table data.
<tr> Table Row The table row container
<td> Table Data The table row container / where your data goes
<th> Table Header the <th>elements define the header of the table.
<tfoot> Table Foot The set of rows defining the footer in a table.
<caption> Caption Provides a title or caption for the table. It should be placed immediately after the opening 'table' tag.
<colgroup> Column Group Group a set of 'col' elements, providing styling options for columns
<col> Column Defines properties for a column in a table when used within a 'colgroup'

Table Attritutes

ATTRIBUTE NAME DESCRIPTION
<colspan> Column Span Defines how many columns a 'td' or 'th' element should span.
<rowspan> Row Span Defines how many rows a 'td' or 'th' element should span.
